HTML

Hypertext Markup Landguage (HTML) is the standard markup language for creating web pages and applications.

Web browsers receive HTML documents from a web server or from local storage and render them into multiledia web pages. HTML describes the structure of a web page semantically and originally included cues for the apperarance of the document.
